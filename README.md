# 杨杨｜游戏原画 & 2D 主美作品集

基于 React、Vite 与 Framer Motion 的个人作品集基础版本。

## 本地预览

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址（通常为 `http://localhost:5173`）。

## 正式构建

```bash
npm run build
npm run preview
```

## 后续替换内容

- `src/main.jsx` 中的 `projects`：替换项目名称、职责和作品封面。
- Hero 的 `<video>`：替换为个人 Showreel 或游戏氛围视频。
- `About` 模块中的抽象人物占位：替换为个人照片或人物设定图。
- 联系方式已按简历录入，请上线前再次核对。
